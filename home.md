## Project Overview

STO Info is a community-driven platform built to support the Star Trek Online player base.

High-level stack:

- Backend: NestJS
- Frontend: Angular
- Database: PostgreSQL
- Hosting: Render
- Edge and DNS: Cloudflare
- CI/CD: GitHub Actions

For a system-level overview, see [Architecture Overview](architecture-overview.md).
