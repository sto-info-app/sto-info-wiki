# Contributing to the STO Info Wiki

Thank you for your interest in contributing.

This repository exists solely to support the central Wiki for STO Info. It acts as a navigation and orientation layer across the project ecosystem.

Canonical technical documentation remains inside each repository's `/docs` directory.

---

## Purpose of This Repository

The Wiki is intended to:

- Provide high-level architecture overview
- Connect repositories and documentation
- Outline roadmap direction
- Document governance principles
- Support onboarding
- Summarise feature concepts

It is not intended to:

- Duplicate technical documentation
- Replace repository `/docs`
- Contain implementation-level detail

---

## Contribution Guidelines

Before submitting changes:

1. Ensure content belongs in the Wiki and not a repository `/docs` folder.
2. Keep content high-level and navigational.
3. Link to canonical documentation rather than copying it.
4. Maintain consistent naming and structure.

---

## Editing Expectations

- All changes should be made via pull request.
- Keep pages concise and structured.
- Avoid adding sensitive operational information.
- Ensure links remain accurate.
